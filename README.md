- [ ] Structure your webpage

## Introduction to HTML

```html
<!DOCTYPE html>
<html lang="fr"> <!-- Webpage language -->
    <head>
        <meta charset="utf-8"> <!-- Page encoding -->
        <title>Page title</title> <!-- Shown on your browser tab -->
    </head>
    <body>
        <!-- Insert the content here  -->
    </body>
</html>
```
